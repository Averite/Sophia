### Japanese 
これは大学生の選択授業システムで、教師管理と管理者管理が含まれています。教師は学生の成績を採点することができます。
学生は授業を選択し、授業を変更することができます。主な技術：Spring、SpringMVC、MyBatis

### English 
This is a university student course selection system, which includes teacher management and administrator management. 
Teachers can grade students' performance. Students can select courses and change courses. Main technologies: Spring, SpringMVC, MyBatis
